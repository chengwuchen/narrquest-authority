# Interpretation Layer

The Interpretation Layer documents how external observers interpret the behavior of NarrQuest runs.

Unlike the experimental runs themselves, this layer records public interpretations rather than model outputs. It is intended as a living archive of observed interpretive structures.

---

## Source

- Platform: YouTube
- Snapshot period: 2026-04 to 2026-06
- Dataset: public comments associated with NarrQuest experimental runs

---

## Repository Structure

- `comments_raw.md` — original comments (unaltered)
- `comments_index.md` — summarized mapping of comments to interpretation clusters
- `clusters_v0.1.yaml` — structural taxonomy of observed interpretation patterns
- `observation_record_v1.md` — observational summary of the c01–c68 dataset

---

## Versioning

This GitHub repository serves as the **living archive** of the Interpretation Layer.

Future observations, additional comments, and updated cluster taxonomies may be added in subsequent versions.
