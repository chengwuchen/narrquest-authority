### Run 002

This run examines how responses behave under strict binary constraints with no allowance for reframing.

The model commits to a single principle and maintains consistent answers across all pressure scenarios.

However, when resolving potential conflicts, it narrows the definition of the chosen principle,
allowing flexibility in execution while preserving surface-level consistency.

This pattern shows that consistency can be maintained through implicit redefinition,
rather than strict adherence to the original constraints.
