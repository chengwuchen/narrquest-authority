### Run 001

This run shows that response stability depends on prompt structure rather than being a fixed property.

Under identical prompts (A), answers remain consistent while reasoning structure drifts and expands.

With small prompt variations (B), stance selection becomes unstable, shifting between competing value priorities.

In cross-domain and sequential contexts (C), local overrides appear, producing inconsistencies across related questions.

In abstract/meta framing (D), the system converges by reframing the problem into a non-binary structure, reducing conflict.

The observed pattern suggests that stability, drift, and convergence are produced locally through prompt framing, domain context, and sequential interaction.
